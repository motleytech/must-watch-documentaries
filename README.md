# must-watch-documentaries

This is a list of documentaries that hopefully will eventually shape up to become a must watch list.

As a starting point, I have copied 300 documentaries from an [excellent source](http://www.illulife.com/blog-spiritual-knowledge-for-enlightenment-for-all-spiritual-entreprenuers/166-documentaries-to-expand-your-consciousness#axzz3u7WTBrMw).

I plan to continuously update this list (even remove some), and also include documentaries suggested by other people (that's why its on github). So, please suggest documentaries that you feel are deserving a place on this list, possibly also stating why (in a brief description).

You can either submit an issue, or create a pull request for submitting your suggestions.
